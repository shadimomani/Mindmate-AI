# MindMate AI 🧠

An AI-powered productivity and self-management platform that helps users 
organize their time, tasks, and mental clarity through intelligent assistance.

## ✨ Features

- **AI Assistant** — Conversational AI to help you think, plan, and execute
- **Smart Authentication** — Secure login via Google or email/password

## 🛠 Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS + shadcn/ui
- **Backend:** Supabase (Auth + Database)
- **AI:** Anthropic Claude API
- **Build Tool:** Vite

## 🚀 Getting Started

```bash
git clone https://github.com/shadimomani/Mindmate-AI.git
cd Mindmate-AI
npm install
npm run dev
```
